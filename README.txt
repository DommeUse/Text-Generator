Эта моя реализация проекта генерации текста.
Файлы проекта:
>Папка data - в ней хранятся все тексты песен и произведений
>generate.py - код, реализующий генерацию предложения(поддерживает аргументы "--model","--prefix","--length" из описания задания)
>train.py - код, создающий модель на основе анализа текстов(поддерживает аргументы "--input_dir","--model" из описания задания)
>model.pkl - собственно сама модель, созданная с помощью библиотеки pickle

Папку data можно спокойно дополнять, то есть код работает независимо от количества и названия файлов.
